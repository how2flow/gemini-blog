# 블로그 페이지 규칙

이 디렉토리 아래에는 각 path들의 얼굴이 되는 페이지 문서가 있습니다.
각 path마다의 블로그 작성 규칙에 따라 문서가 업데이트 되었을 경우, 반드시 페이지에
수정사항이 필요한지 판단해야 합니다.

```
.
├── about.md
├── category-archive.md
├── diary.md
├── documents.md
├── GEMENI.md
├── help.md
├── legacy.md
├── posts.md
├── posts-translate.md
└── tag-archive.md
```
